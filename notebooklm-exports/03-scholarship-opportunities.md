## SUCCESS TIPS

### Application Best Practices
1. **Start Early**: Don't wait until deadline week
2. **Reuse Essays**: Adapt core essays for multiple applications
3. **Request Recommendations Early**: Professors need 2-3 weeks
4. **Proofread**: Have someone review all applications
5. **Follow Instructions**: Exactly as stated
6. **Track Deadlines**: Use calendar reminders
7. **Save Receipts**: Confirmation emails for all submissions

### Essay Themes to Emphasize
- **Women in STEM**: Overcoming barriers, passion for AI/Data Science
- **Study Abroad**: EU citizenship, trilingual skills, cultural bridge
- **Technical Skills**: Python/REACT projects, coding portfolio
- **Career Goals**: Clear vision for AI/Data Science career
- **Language Advantage**: German-English-Spanish as career asset

### Common Application Components (Prepare Once)
- Resume (1-page, technical focus)
- Transcripts (official and unofficial)
- Personal statement (500-1000 words)
- Why Tech/AI essay (300-500 words)
- Recommendation letters (2-3 professors/mentors)
- Financial documents (for need-based scholarships)
- FAFSA (required for Gilman)
